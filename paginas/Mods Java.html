Mods Para minecraft java (la mayoria de la 1.12.2) que mucha gente no conoce y son INCREIBLES


1.- SecurityCraft: este mod se hizo conocido por la serie de YT/Twitch Karmaland, este mod lo que agrega a tu mundo o server, son mas objetos de pretoccion para tu casa, como puede ser, un puerta con scanner de ojos, o torretas que disparan a otros jugadores que se acerquen a tu casa, camaras de seguridad, etc, aqui dejo el link del mod: Descargar SecurityCraft (aqui les dejo un consejo, cuando esten en la pagina de curseforge vayan a la pestaña archivos, ahi eliges la version del mod, gracias)

2.- ChocoCraft: este mod agrega una especie de animal que tienes que domesticar, con estos animales puedes volar de forma libre por el mapa una vez domesticados claro esta, pero al volar se gasta la energia del chocobo, este mod tambien se hizo conocido por la serie de YT/Twitch Karmaland aqui les dejo el link del mod: Descargar ChocoCraft (aqui les dejo un consejo, cuando esten en la pagina de curseforge vayan a la pestaña archivos, ahi eliges la version del mod, gracias)



3.- MrCrayfish Vehicle mod: este mod como lo dice en su nombre trata de agregar automobiles a minecraft, este mod sirve para tranportarse por el mapa, ideal para Roleplays, si quieres aprender a crearte un server roleplay aqui pongo un video Video para hacer un server Roleplay Aqui el link de el mod: Descargar MrCrayfish Vehicle mod

4.- MrCrayFish Gun Mod: Este mod como lo dice en su nombre trata de agregar armas a minecraft, este mod es de los mas livianos que encontre porque hay otros que hace que tu computadora explote, tu computadora del año de la papa, lo corre sin problemas :D aqui el link para descargar el mod: Descargar MrCrayFish Gun Mod

5.- DecoCraft: este mod agrega variedad de cosas para mejorar y adornar tu casa de minecraft, este mod trae, mesas, sillas, muebles de cocina y muebles normales, aqui les dejo el link del mod: Descargar DecoCraft

5.- Emoticons Mod: este mod hace que tengas animaciones al caminar, volar, correr, agacharte, pero lo mas sorprendente es que PUEDES HACER BAILES DE FORTNITE este mod nesesita de otro mod porque si no este no funcionara, aqui dejo los links para los 2 mods: Descargar emoticons =====> Descargar McLib (necesario para el emoticons)

6.- Chisel: este mod agrega bloques para realizar diferentes contrucciones, con una herramienta del mod puedes modificar esos bloques, este mod se hizo conosido por la serie de YT/twitch Karmaland aqui dejo el link del mod: Descargar Chisel

7.- Dynamic trees: Este mod se hizo conocido por la serie de YT/twitch Karmaland, este mod sirve o aporta que los arboldes tienen una animacion realista al cortarlos, cayendo y asi hacer que el jugador farmee madera mas facil y sencillo, aqui les dejo el link del mod: Descargar Dynamic Trees

8.- Minecraft Comes Alive (MCA): este mod trata de podre formar una familia, tener trabajadores, y tener hijos 7u7, este mod se hizo famoso gracias a planeta vegetta, aqui dejo el link del mod: Descargar MCA

9.- Custom Npc's: Este mod tiene como utilidad el poder configurar tu propios Npc, si no sabes que son los npc, son "bots" del minecraft y tu puedes configurarlos como gustes por ejemplo: quieres un vendedor de bloques, tu lo configuras para que venda los bloques que tu le digas, aqui les dejo el link del mod: Descargar Custom Npc's
